# 🎰 Slot Machine Game (Python)

A simple command-line slot machine betting game built using Python.

## 🚀 Features
- Deposit system
- Multiple betting lines (1–3)
- Adjustable bet per line
- Random slot generation
- Winning calculation logic

## 🛠 Tech Used
- Python
- Random module

## 🎮 How to Run

1. Clone the repository:
   git clone https://github.com/yourusername/slot-machine-game.git

2. Navigate to the project folder:
   cd slot-machine-game

3. Run the game:
   python main.py
